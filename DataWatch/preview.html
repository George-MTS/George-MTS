DataWatch

Real-time cellular & Wi-Fi data usage tracker built for Safaricom users in Kenya. Everything stays on your iPhone — encrypted, private, zero servers.

📊

Live Dashboard

Top apps ranked by data usage. Background vs. foreground split. Progress bar per app. Cellular and Wi-Fi totals side by side.

📈

App Breakdown Charts

Hourly usage chart for today. 7-day cumulative view. Foreground/background percentage split. Built with native SwiftUI — no chart libraries.

🔔

Smart Local Alerts

Notifies when any single app exceeds 50 MB or your daily threshold is hit. Local push only — no external server ever touched.

🔒

AES-256 Privacy

All CoreData records encrypted before writing. Key stored in iOS Keychain. Entitlements explicitly block outbound network access.
